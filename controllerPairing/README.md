# About

A tool for viewing and setting the bluetooth address
a sixaxis controller is currently paired with.


# Dependencies

HID API (http://www.signal11.us/oss/hidapi/)


# Supported Platforms

* Windows
* Mac
* Linux


# Building

    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    $ ./bin/sixaxispairer
    $ ./bin/sixaxispairer xx:xx:xx:xx:xx:xx
